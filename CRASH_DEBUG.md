# Crash Debug Instructions

To help diagnose the crash, please:

1. Open Android Studio
2. Go to View → Tool Windows → Logcat
3. Filter by "BookBuddy" or search for "FATAL"
4. Run the app
5. When it crashes, copy the ENTIRE error message including:
   - The FATAL EXCEPTION line
   - The stack trace
   - Any "Caused by" messages

Please paste the complete error here.
